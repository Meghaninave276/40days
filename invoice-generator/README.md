# Invoice Generator

A modern, professional React-based invoice generator web app with real-time calculations, dynamic items, tax, and PDF export.

## Features
- Responsive design with Tailwind CSS
- Dynamic item list (add/remove)
- Real-time subtotal, tax, grand total calculations
- Currency selector (USD, EUR, INR, GBP)
- PDF export with jsPDF
- Print support
- Clean SaaS-style UI

## Quick Start
```bash
npm install
npm run dev
```

Open http://localhost:5173

## Usage
1. Fill company and customer details
2. Set invoice number, dates
3. Add items with qty/price
4. Adjust tax rate
5. Download PDF or Print

## Libraries
- React 19
- Tailwind CSS
- jsPDF
- Heroicons

Built with Vite.
